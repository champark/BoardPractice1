라이언 2020. 1. 7. 14:10
반응형

실행 잘되던 프로젝트에 maven - update project... 를 클릭후 정상적으로 실행이 안되서 찾아보니

프로젝트 우클릭 - Maven - Update Project 실행시 maven 라이브러리 경로가 삭제되는 현상이 발생한다고 한다.

 

문제 해결 방법

이 문제를 해결하기 위해서 라이브러리 경로를 다시 설정해주면 된다.

 

  • 프로젝트 우클릭 - properties
  • Deployment Assembly - Add 버튼 - Java Build Path Entries - Maven Dependencies 선택 - Apply버튼
